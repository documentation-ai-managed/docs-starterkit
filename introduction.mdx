## Where content comes from

- Pages live in `.mdx` files (like this one).
- Navigation is defined in `documentation.json`.
- API Reference is generated from `openapi.yaml` when you choose to use it.
- Read the [Sample Notice](/sample-notice) before sharing this site.

## Make your first edit

<Steps>
  <Step title="Rewrite this page" icon="edit">
    Update the title, description, and sections in `introduction.mdx`.
  </Step>
  <Step title="Customize Quickstart" icon="zap">
    Replace the steps in `quickstart.mdx` with your real onboarding flow.
  </Step>
  <Step title="Add your first page" icon="file-text">
    Create a new `.mdx` file and register it in `documentation.json`.
  </Step>
</Steps>
