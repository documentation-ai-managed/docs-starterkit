## What is StarterKit?

StarterKit provides a complete foundation for building AI-powered documentation sites with Documentation.AI. You get pre-configured components, themes, and integrations that let you focus on content instead of setup. Deploy your docs in minutes and scale effortlessly as your project grows.

<Callout kind="info">
  StarterKit uses modern MDX and React components to create interactive, searchable documentation.
</Callout>

## Quick Start

Get up and running with StarterKit in under 5 minutes.

<Steps>
  <Step title="Clone and Install" icon="download">
    Start by cloning the repository and installing dependencies.

````bash
git clone https://github.com/documentation-ai/starterkit.git
cd starterkit
npm install
````
  </Step>
  <Step title="Configure AI Key" icon="settings">
    Add your Documentation.AI API key to the environment.

    Create a `.env.local` file:

````javascript
DOCUMENTATION_AI_API_KEY=your-api-key-here
````
  </Step>
  <Step title="Run Development Server" icon="play">
    Launch the dev server and open your browser.

````bash
npm run dev
# Visit http://localhost:3000
````
  </Step>
</Steps>

## Multi-Language Support Example

Showcase code in different languages effortlessly.

<CodeGroup tabs="JavaScript,Python">
  ```javascript
  // Generate docs via API
  const response = await fetch('https://api.example.com/v1/docs/generate', {
    method: 'POST',
    headers: { 'Authorization': `Bearer ${YOUR_API_KEY}` },
    body: JSON.stringify({ codebase: 'path/to/repo' })
  });
  const docs = await response.json();
  ```
  ```python
  # Generate docs via API
  import requests
  response = requests.post(
      'https://api.example.com/v1/docs/generate',
      headers={'Authorization': f'Bearer {YOUR_API_KEY}'},
      json={'codebase': 'path/to/repo'}
  )
  docs = response.json()
  ```
</CodeGroup>
