## Overview

StarterKit helps you quickly create new Documentation.AI projects.

## Steps

<Steps>
  <Step title="Create a folder">
    ````bash
    mkdir my-docs && cd my-docs
    ````
  </Step>
  <Step title="Initialize project">
    ````bash
    npm init -y
    ````
  </Step>
  <Step title="Install StarterKit">
    ````bash
    npm install @documentation-ai/starterkit
    ````
  </Step>
</Steps>
