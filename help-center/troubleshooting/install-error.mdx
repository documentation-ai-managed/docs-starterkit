---
title: Page Missing in Sidebar
description: Fix a page that does not appear in navigation.
---

## Symptoms

- A new page file exists but does not show up in the sidebar.
- Clicking a nav item shows a 404.

## Likely causes

- The page path is missing in `documentation.json`.
- The path does not match the `.mdx` filename.

## Fix

<Steps>
  <Step title="Confirm the file name" icon="file-text">
    Ensure the file exists (example: `my-page.mdx`).
  </Step>
  <Step title="Update navigation" icon="settings">
    Add or fix the matching `path` in `documentation.json`.
  </Step>
  <Step title="Refresh preview" icon="refresh-cw">
    Reload your preview so changes take effect.
  </Step>
</Steps>
