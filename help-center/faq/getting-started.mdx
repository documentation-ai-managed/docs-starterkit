## Overview

To use StarterKit with Documentation.AI, install it via npm or yarn.

## Installation Steps

1. Open your terminal
2. Navigate to your project folder
3. Run one of the following:
   
```bash
npm install @documentation-ai/starterkit
# or
yarn add @documentation-ai/starterkit
```

StarterKit is now ready to use in your documentation project.
